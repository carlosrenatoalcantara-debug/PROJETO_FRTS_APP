🔐 ADICIONAR NOVA API GEMINI (SEGURA)
───────────────────────────────────────────────────────────────────────────

Passo 1: Gerar nova chave em https://ai.google.dev/gemini-api

Passo 2: Fazer login e obter JWT token:
  curl -X POST http://localhost:3000/api/auth/login \
    -H "Content-Type: application/json" \
    -d '{
      "email": "admin@fortesolar.com.br",
      "password": "AdminPass123!"
    }'

Passo 3: Adicionar chave (CRIPTOGRAFADA no BD):
  curl -X POST http://localhost:3000/api/integrations/add-key \
    -H "Authorization: Bearer <JWT_TOKEN>" \
    -H "Content-Type: application/json" \
    -d '{
      "integrationName": "GeminiVision",
      "apiKey": "<sua_nova_chave_aqui>"
    }'

✅ Chave será armazenada:
   • Criptografada (AES-256-GCM)
   • Com salt único
   • Com auditoria completa
   • Com rotação automática (90 dias)
