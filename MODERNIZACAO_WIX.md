## 🎯 Mudanças Específicas no Wix

### 1. Editar Hero Section
1. No editor, clique na imagem de fundo (painéis solares)
2. Considere usar uma imagem mais moderna (painéis solares em telhado residencial)
3. Ajuste o contraste para melhor legibilidade
4. Atualize o título e descrição com os textos novos

### 2. Adicionar Seção de Benefícios
1. Clique em "Adicionar" seção
2. Escolha "Layout de 4 Colunas"
3. Adicione 4 cards com os benefícios
4. Use cores: Azul (#0066CC), Laranja (#FF6600), Verde (#00AA44)

### 3. Criar Seção "Como Funciona"
1. Adicione nova seção
2. Use elemento "Timeline" ou "Steps"
3. Adicione os 4 passos com ícones

### 4. Integrar Calculadora
1. Adicione botões em destaque
2. Link direto: https://projeto-frts-app.vercel.app/calculadora
3. Coloque em locais estratégicos:
   - Hero section (CTA principal)
   - Seção calculadora
   - Menu de navegação ("Simulador")

### 5. Atualizar Menu
Estrutura recomendada:
```
Início → Sobre → Como Funciona → Simulador → Orçamento → Contato
```

## 🎨 Dicas de Design Moderno

### Cores Recomendadas
- **Primária:** Azul (#0066CC ou #0052A3) - Confiança
- **Secundária:** Laranja (#FF6600 ou #FF8C00) - Energia
- **Terceira:** Verde (#00AA44 ou #00B359) - Sustentabilidade
- **Neutra:** Cinza claro (#F5F5F5) - Fundos

### Tipografia
- **Títulos:** Font Sans-serif moderna (ex: Montserrat, Inter, Poppins)
- **Corpo:** Font legível (ex: Open Sans, Roboto)
- **Tamanho mínimo:** 16px para melhor leitura mobile

### Espaçamento
- Margens generosas entre seções
- Padding de 40-60px em seções
- Margem inferior de 20px entre elementos

### Mobile First
- Certifique-se que tudo funciona em smartphones
- Botões com tamanho mínimo de 48x48px
- Textos legíveis em telas pequenas
