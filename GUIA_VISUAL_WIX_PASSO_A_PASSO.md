# 📸 Guia Visual Passo-a-Passo: Modernizar Wix

## 🎯 Objetivo Final
Seu site Wix terá:
- ✅ Hero moderno com novo título

- ✅ Seção "Como Funciona" com 4 passos

- ✅ Links funcionando

**Tempo total: 45-60 minutos**

## 📍 PASSO 1: Abrir o Editor Wix

### 1.1 - Acessar o Site
```
1. Abra o navegador (Chrome, Firefox, Edge)
2. Vá para: www.wix.com
3. Faça login com suas credenciais Wix
```

**Você verá a tela inicial com seus sites**

### 1.2 - Selecionar o Site Forte Solar
```
1. Procure o site "Forte Solar" ou "fortesolar.com.br"
2. Clique nele
```

**Agora você está na dashboard do site**

### 1.3 - Abrir o Editor
```
1. Procure um botão chamado "Editar site" ou "Edit"
2. Clique nele
   └─ Espere alguns segundos para carregar
```

**Pronto! Você está no editor Wix!**

Você verá:
- Sua página com o layout atual
- Menu à esquerda com opções
- Menu superior com "Publicar", "Visualizar", etc.

---

## 📍 PASSO 2: Adicionar o Código das Novas Seções

### 2.1 - Navegue até o Final da Página

```
1. No editor, role a página para BAIXO
2. Vá até o final (antes do footer)
3. Você deve estar vendo a última seção atual
```

**Dica:** Use a barra de rolagem direita para descer mais rápido

### 2.2 - Adicionar um Bloco de Código

```
1. Clique em "Adicionar" (menu esquerdo superior)
   └─ Você verá opções: Layout, Elementos, Média, etc.

2. Procure por "Código" ou "HTML Embed"
   └─ Se não achar, vá em "Mais" → "Código Customizado"

3. Clique para adicionar
```

**Agora você tem um novo bloco em branco**

### 2.3 - Colar o Código

```
1. Abra o arquivo: WIX_CUSTOM_CODE.html
   └─ Está na pasta: PROJETO_FRTS_APP

2. Selecione TODO o texto (Ctrl+A)

3. Copie (Ctrl+C)

4. Volte ao editor Wix

5. Clique no novo bloco de código que criou

6. Cole o código (Ctrl+V)
```

**Pronto! O código foi adicionado!**

Você verá um preview das 4 novas seções aparecendo:
- Benefícios (4 cards)
- Como Funciona (4 passos)
- CTA Calculadora
- CTA Final

## 📍 PASSO 3: Atualizar a Seção Hero

A seção Hero é a primeira seção da sua página (título + imagem + botão).

### 3.1 - Localizar o Hero Section

```
1. Role para CIMA na página
2. Você verá a primeira seção (com imagem grande)
3. Este é o seu Hero Section
```

### 3.2 - Editar o Título

```
1. Clique no título principal da página
   └─ Deve estar escrito algo como "A força do sol..."

2. Apareça um campo para editar

3. Apague tudo e digite:
   "Energia Solar Confiável para Sua Casa"

4. Clique fora ou pressione Enter
```

**Seu novo título está pronto!**

### 3.3 - Editar a Descrição

```
1. Clique no texto de descrição abaixo do título
   └─ É o parágrafo que começa com "A Forte Solar é..."

2. Apague tudo e copie-cole este texto:

"Transforme sua conta de energia em economia real. Sistemas solares 
de alta eficiência, instalação profissional e suporte completo. Sua 
casa merece energia limpa, abundante e econômica. Descubra quanto você 
pode economizar."

3. Clique fora
```

**Descrição atualizada!**

### 3.4 - Atualizar o Botão CTA

```
1. Clique no botão da seção hero
   └─ Procure um botão que diz algo tipo "Saiba mais" ou similar

2. Mude o texto para:
   "Calcule sua Economia ☀️"

3. Agora configure o link:
   - Clique no ícone de link/engrenagem (deve aparecer)
   - Digite ou cole a URL:
     https://projeto-frts-app.vercel.app/calculadora
   - Salve

4. Clique fora
```

**Button atualizado com link funcionando!**

## 🆘 Solução de Problemas

### "Não consigo encontrar 'Adicionar' no Wix"
```
- Procure pelo ícone "+" no lado esquerdo
- Ou clique na área vazia da página onde quer adicionar
```

### "O código não está aparecendo"
```
1. Verifique se colou TUDO o código
2. Clique no bloco de código no editor
3. Se vazio, apague e tente colar novamente
4. Use Ctrl+A depois Ctrl+V para garantir
```

### "Os links não funcionam"
```
1. Clique no bloco de código
2. Procure pelos links:
   - https://projeto-frts-app.vercel.app/calculadora
   - https://wa.me/5584994047722
3. Verifique se estão corretos
4. Se o número WhatsApp for diferente, atualize
```

### "O design fica diferente no mobile"
```
1. Volte ao editor
2. Clique no bloco de código
3. Procure por @media (max-width: 768px)
4. Você pode ajustar tamanhos de fonte, padding, etc.
```

### "As cores não estão iguais"
```
Cores do código:
- Azul: #0066CC (títulos)
- Laranja: #FF6600 (botões principais)
- Verde: #00AA44 (banner final)
- Cinza claro: #F5F5F5 (fundos)

Se quiser mudar, procure no bloco de código:
--primary: #0066CC;
--secondary: #FF6600;
--success: #00AA44;
```
