## KEY IMPLEMENTATION FEATURES

### 1. Deterministic Parsing
- All extraction uses regex patterns with consistent ordering
- No random elements or temporal dependencies
- Parsing results are reproducible from same PDF
- Confidence scoring is based on data completeness (deterministic)

### 2. Immutable Outputs
- All DTO functions return `deepFreezeSafe()` objects
- Extracted data cannot be mutated at runtime
- Frozen structure prevents accidental modifications
- Alignment with v2.1_EV_INTEGRATED immutability contract

### 3. Brazilian Locale Support
- Handles both CPF and CNPJ formats
- Converts Brazilian numeric format: 1.234,56 → 1234.56
- Normalizes state codes to uppercase (SP, RJ, etc.)
- Recognizes regional address patterns

### 4. Resilience & Error Handling
- Graceful handling of missing fields
- Null returns for unavailable data (vs. crashes)
- Clear error codes for each failure scenario
- Partial extraction possible without crashes
- Confidence scoring indicates data quality

### 5. Extensibility
- Parser architecture supports multiple distributors
- Factory pattern (`createNeoenergiaParser()`) for easy extension
- DTO layer independent of parser implementation
- Future distributors can be added without modifying existing code

## EXTRACTION CONFIDENCE SCORING

**Scoring Model** (100% baseline):
- Missing nome_cliente: -15%
- Missing cpf_cnpj_parcial: -10%
- Missing endereco: -10%
- Missing cidade: -10%
- Missing estado: -5%
- Missing codigo_instalacao: -20% (CRITICAL)
- Missing codigo_cliente: -15%
- Missing consumo_medio: -15%

**Interpretation**:
- 85-100%: High confidence, ready for project creation
- 70-85%: Medium confidence, review before creation
- <70%: Low confidence, manual correction required

## IMMUTABILITY CONTRACT

All extracted data is frozen via `deepFreezeSafe()`:

```javascript
const resultado = await billIntakeService.processBill(...)
// resultado.extraido is now immutable
resultado.extraido.cliente.nome_cliente = "NOVO"  // TypeError!
resultado.extraido.consumo.consumo_medio = 9999   // TypeError!
```

**Guarantee**: Engineering truth from bill extraction cannot be mutated at runtime.
